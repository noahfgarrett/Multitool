Interaction Details

  • Checkboxes — Select one or multiple entries. Bulk indent/outdent applies to selection.
  • Tab / Shift+Tab — Indent or outdent selected entries (makes them children or promotes to parents).
  • Drag handles — Reorder entries within their level.
  • Pencil icon / Click name — Inline edit the description text.
  • Three-dot menu — Add child entry, delete, duplicate.
  • "Auto-detect Names" — Scans pages via pdfjs text extraction, updates labels. Shows (auto) tag on fallback names.
  • "Add Entry" — Manually add a new entry (prompts for page number).
  • No. column — Auto-calculated from numbering preset + hierarchy level.
  • Pages column — Shows page count for that section (parent = sum of children).
  • Page column — Starting page number in the merged document.
