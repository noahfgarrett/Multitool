Design Notes

  • Type selector — toggle buttons instead of dropdown for the two options (more visual, faster)
  • Tool dropdown — populated from registry.ts + "New Tool Idea" + "General / App-wide"
  • Priority — pill buttons (Low/Medium/High), defaults to Medium
  • Conditional help text — description hint changes based on Bug vs Enhancement selection
  • User identity — pulled from their saved profile (name, email, initials), shown as a "Sending as" badge
  • Submit action — opens mailto: with structured body, also tries ms-outlook:// for Outlook users
  • Sidebar button — below all tool categories, above version number, subtle LotusWorks orange accent
