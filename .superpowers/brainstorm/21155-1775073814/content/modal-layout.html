Behavior

  • Default tab: "Update" when there's an update available; "Changelog" if opened from a menu/button
  • Releases collapsed by default — click a version header to expand its notes
  • Latest release auto-expanded — so the user immediately sees what's new
  • "See all releases" — expands the modal to full height and shows every release
  • Orange left-border on the latest release, subtle border on older ones
  • "latest" badge on the newest version
  • Scrollable — max-height with overflow-y on the changelog list
